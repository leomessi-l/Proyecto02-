Fondos: asignamos un fondo negro (#000000) para el cuerpo de la página, dándole un aspecto un poco mas serio.
Clase .hero-critico: Crea una sección principal con un fondo rojo bastante fuerte (#ce1212), color que suele ser vinvulado con el escándalo.
Clase .card-polemica: Añade un borde negro a la izquierda de las tarjetas para destacarlas.
Navbar (seria barra de navegación): Situada en la parte de arriba con clase sticky-top Incluye un hashtag en amarillo para llamar la atención.
Hero Section (Encabezado): Una sección de gran tamaño que presenta el tema central ("¿El fin del fútbol?") con un botón de acción (CTA) llamativo para "Firmar Petición".
Alert (Alerta): Se usa un alert-danger (el cuadro rojo) para presentar la denuncia principal de forma urgente.
Sistema de Rejilla (row y col-md-4): Organiza el contenido en tres columnas en pantallas grandes, que se apilan verticalmente en móviles.
Cards (Tarjetas): Utilizadas para presentar puntos específicos (como el tema de los "Torneos Insólitos"). Incluyen una sombra (shadow-sm) para dar profundidad.
List Groups (Listas): Se usa para enumerar de forma ordenada los clubes que el autor considera "amigos del poder".
Badges (Etiquetas): Una pequeña etiqueta roja con el texto "Escándalo" para jerarquizar la información.
Elementos Visuales y Multimedia
Imágenes Externas (<img>): Se insertan fotos de Claudio Tapia y situaciones polémicas mediante URLs externas para dar contexto visual a las críticas.
Colores:
Negro, Rojo y Blanco: Una combinación agresiva que refuerza el mensaje de protesta.
Amarillo (btn-warning): Usado en botones y hashtags importantes para que resalten sobre el fondo oscuro.
